# Todo List
App para mastrar e criar tarefas.
Este app foi desenvolvido como projeto final dentro do Bootcamp Mobile Santander 2021

## Principais recursos usados

 - Arquitetura MVVM
 - RecycleView
 - Room
 - DataBinding
 - LiveData
 - ViewModel
 - Navigation
